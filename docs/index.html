Full Visibility in One Command

/skill-monitor security

Security Scan

Pattern-based detection: reverse shells, credential+network combos, sensitive path access, unconfirmed settings.json modifications.

/skill-monitor cost

Token Cost Breakdown

Accurate extraction for both single-line and multi-line YAML descriptions. User vs plugin breakdown. Top 10 most expensive.

/skill-monitor budget

Budget Analysis

Based on official Claude Code spec: 1% context window budget, 250-char truncation, quality assessment for skill count.

/skill-monitor hooks

Hook Conflicts

Scans settings.json and skill frontmatter for duplicate matchers that cause redundant processing.

/skill-monitor overlap

Cross-Layer Overlap

LLM-powered detection of user skills that duplicate plugin capabilities. Recommends remove, keep, or merge.

/skill-monitor compress

Description Optimizer

Shorten descriptions to keep trigger keywords within the 250-char visibility window.

Get Started

Recommended Plugin Install

claude plugin add luxuzhou/skill-monitor-plugin

Git Clone

git clone https://github.com/Luxuzhou/skill-monitor-plugin.git
cp -r skill-monitor-plugin/skills/skill-monitor ~/.claude/skills/

Then Run

/skill-monitor

Optional: /skill-monitor setup-tracking to enable usage tracking.
